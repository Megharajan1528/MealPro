# MealPro

MealPro is a food delivery web application developed using Django. The application allows users to register, log in, browse restaurants, view menus, add items to a cart, and place orders.

## Features

* User Registration and Login
* Restaurant Management
* Menu Management
* Food Ordering
* Shopping Cart
* Customer Dashboard
* Admin Dashboard

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS

## Project Structure

```text
MealPro/
├── myapp/
├── delivery/
├── Templates/
├── static/
├── manage.py
└── db.sqlite3
```

## How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Megharajan1528/MealPro.git
```

2. Navigate to the project folder

```bash
cd MealPro
```

3. Install dependencies

```bash
pip install django
```

4. Run the server

```bash
python manage.py runserver
```

5. Open the browser and visit

```text
http://127.0.0.1:8000/
```
